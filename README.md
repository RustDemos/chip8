# install

```
git clone https://git.sr.ht/~codetyphon/chip8
cd chip8
cargo run ./ROM/TANK
```

## keyboard

Chip8 keyboard:

|     |     |     |     |
| --- | --- | --- | --- |
| 1   | 2   | 3   | C   |
| 4   | 5   | 6   | D   |
| 7   | 8   | 9   | E   |
| A   | 0   | B   | F   |

Is mapped to: 

|     |     |     |     |
| --- | --- | --- | --- |
| 1   | 2   | 3   | 4   |
| Q   | W   | E   | R   |
| A   | S   | D   | F   |
| Z   | X   | C   | V   |
